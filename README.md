# ProjetoGirotto
# Descrição
Um arcade com 3 minijogos dentro dele.
O primeiro é um jogo de perguntas e respostas, para testa seus conhecimentos gerais.
O segundo é um jogo de sorte e aposta. 
O terceiro é um jogo de estratégia.
